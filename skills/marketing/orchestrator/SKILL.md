# Orchestrator Skill

## Overview

Orchestrator is the meta-skill that asks diagnostic questions, diagnoses your needs, and sequences the other 9 skills in the right order. It's the router that ensures you're using the right skill at the right time.

**Keywords**: marketing strategy, skill sequencing, marketing planning, strategy optimization, marketing diagnosis, strategic planning

## Core Methodology

Orchestrator works by:

1. **Understanding Your Situation** — What's your current state?
2. **Diagnosing Your Needs** — Which skills will have highest impact?
3. **Sequencing Strategically** — In what order should you use them?
4. **Executing Systematically** — How do you implement?
5. **Measuring Results** — How do you know if it's working?

## Orchestrator Decision Tree

Use this to know when to move to the next skill:

```
Is my current skill working?

YES:
  ├─ Am I seeing results?
  │  ├─ YES → Continue 2-4 more weeks, then move to next
  │  └─ NO → Adjust and try again 1-2 more weeks
  │
  └─ Do I have time for another skill?
     ├─ YES → Add next skill while continuing current
     └─ NO → Master current first, then move to next

NO:
  ├─ Is it a skill issue or execution issue?
  │  ├─ SKILL → Review framework, adjust approach
  │  └─ EXECUTION → Get help, refine execution
  │
  └─ Should I move to a different skill?
     ├─ YES → Switch to higher-impact skill
     └─ NO → Spend more time mastering this skill
```

## Common Pitfalls

**Trying All Skills at Once** — Dilutes focus. Pick 2-3 first.  
**Wrong Sequence** — Using skills out of order wastes time.  
**Skipping Foundation** — Brand Voice and Positioning are foundation.  
**Not Measuring** — Can't improve what you don't measure.  
**Giving Up Early** — Most skills take 4-8 weeks to show results.
